## 4. リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| ユーザーが課金しない | 収益ゼロ | テンプレ品質を「金を払う価値がある」レベルに |
| SEOアルゴリズム変動 | PV半減 | メール配信・SNSで直接接点を確保 |
| J-OSLER制度変更 | コア商品の価値消失 | 臨床ツール群（制度非依存）を早期構築 |
| 医療情報の誤り | 炎上・信頼失墜 | 免責表示、原著論文確認、誤り報告フォーム |
| 継続モチベーション | 更新停止 | 小さなタスクに分割、こまめに成果を実感 |
