# iwor.jp ビジネス概要

> 最終更新: 2026年3月15日（ドメイン決定・全面改訂）

## ブランド

- **名称**: iwor（イウォル）
- **ドメイン**: iwor.jp（2026-03-15 取得、Xserverドメイン）
- **由来**: アイヌ語で「恵みの地」。神々が住む世界であり、人が生きるために必要なものを自然から受け取る場所。
- **将来**: iwor.com は $9,900 で取得可能（売上が立ってから検討）

### リード獲得ファネル（FREE → PRO転換装置）

**専門科診断（/diagnosis/specialty/）:**
```
Google検索「研修医 何科」「専門科 決められない」
  ↓
iwor.jp/diagnosis/specialty/（無料・診断開始）
  ↓
20問クリック式（性格・興味・QOL・手技好み・勤務地等）
  ↓
「結果を表示するには iwor PRO に登録」→ 年9,800円
  ↓
結果＋詳細レポート＋おすすめプログラム表示
  ↓
PRO継続（学習ダッシュボード＋病棟TODO＋論文フィード）
```

**マーケ価値**: 回答データ＝研修医のキャリア志向・勤務地・経歴。売却時に「研修医9,000人/年のデータ資産」として評価される。

**初期研修 学習ダッシュボード（/learning/residency/）:**
- 学習項目一覧はFREE（「研修医が身につけるべきこと」でSEO）
- 進捗管理・パーソナライズはPRO
- 完了率○%が表示される → 「研修の勉強＝iwor」の口コミ狙い

## 競合分析サマリー

### 業界共通パターン
m3/HOKUTO/ケアネット/メドピア/Antaa → 全社「医師を無料で集めて製薬企業に広告を売る」モデル
→ 構造的に医師への直接課金ができない
→ **iworの構造的優位性: 医師への直接課金が可能**

### HOKUTOの弱点（iworが突ける）
- 500種ツールをアプリに囲い込み → iworはWebで完全公開
- 製薬広告モデル上、Web完全公開が構造的にできない
- Web検索に出ないアプリ内ツール ＝ SEO空白地帯
