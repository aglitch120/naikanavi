## 1. エグゼクティブサマリー

**iwor（イウォル）** は、医師の臨床とキャリアを支える日本語Webプラットフォームです。

- **ドメイン:** iwor.jp（2026年3月取得）
- **コンテンツ:** 臨床計算ツール36個 + ブログ記事173本
- **ターゲット:** 内科専攻医・研修医・医学生（将来的に全診療科）
- **収益モデル:** フリーミアム（無料ツール → PRO年額9,800円）
- **技術スタック:** Next.js + Cloudflare Pages（月額コスト約100円）
- **フェーズ:** MVP完成・SEO集客開始段階

**事業の強み:**
1. HOKUTOのアプリ内ツールがWeb検索に出ない空白地帯を獲るSEO戦略
2. 36個の臨床計算ツール（SEO集客装置として機能）
3. 173本のJ-OSLER・専門医試験コンテンツ（ロングテールSEO資産）
4. 月額100円のインフラコスト（高い利益率ポテンシャル）
5. 単独運営可能な技術アーキテクチャ（属人性低）

## 3. 市場分析

### 3.1 TAM / SAM / SOM

|------|------|------|
| **TAM** | 日本の医師・医学生全体 | 約35万人（医師32万人 + 医学生5万人） |
| **SAM** | 内科系医師 + 研修医 + 専攻医 | 約8万人 |
| **SOM** | J-OSLER利用者 + 内科専門医試験受験者 | 約2万人/年 |

- **口コミ伝播:** 医局・研修病院単位での紹介が期待
- **季節性:** 4月（新年度開始）・10月（症例登録ラッシュ）にピーク

## 4. 競合分析

### 4.1 競合マップ

| 競合 | 強み | 弱み | iwor差別化 |
|------|------|------|-----------|
| **HOKUTO** | アプリUI、資金力、医師認証 | ツールがWeb検索に出ない | SEOでWeb検索を獲る |
| **m3** | 医師会員32万人、転職事業 | 臨床ツールは手薄 | 臨床ツール特化 |
| **メドピア** | 薬剤口コミDB | J-OSLER機能なし | 専攻医特化 |
| **ケアネット** | 動画教育コンテンツ | 計算ツールなし | ツール+コンテンツ統合 |
| **今日の臨床サポート** | 包括的データベース | 高額・法人契約中心 | 個人向け無料ツール |

### 4.2 競合優位性（Moat）

1. **SEO空白地帯:** HOKUTOのツールはアプリ内にありGoogleに出ない。iwor.jpが「CHA2DS2-VASc 計算」等のキーワードを獲れる
2. **コンテンツ資産:** J-OSLER特化173記事（専門性が高く競合参入しにくい）
3. **低コスト構造:** Cloudflare Pages無料枠で運営、月額100円
4. **データ資産:** ツール利用ログ・ユーザー属性（将来のマネタイズ基盤）
