## Jobs-to-be-Done 分析

### iworが勝つJob

| Job | 現状の解決策 | iworの優位性 |
|-----|------------|-------------|
| 当直中にスコア計算したい | HOKUTO（アプリ起動が必要）/ Google検索 | Web即アクセス。SEOで上位表示 |
| J-OSLER/EPOCを効率管理したい | Excel / MOTiCAN（iOS限定） | Web+全プラットフォーム+同期ベンチマーク |
| 医学知識を効率的に復習したい | Anki（設定が複雑）/ medu4（映像のみ） | FSRS+日本語UI+コミュニティデッキ |
| マッチング対策を一元化したい | 各病院HP / 先輩の口コミ | 病院DB+志望リスト+人気ランキング+履歴書生成 |
| 全部を1つにまとめたい | HOKUTO+Anki+Excel+各種アプリ | 全部入り¥980。アプリ疲れの解消 |

### iworが戦わないJob（意図的に避ける）

| Job | 支配的競合 | 避ける理由 |
|-----|-----------|-----------|
| 臨床情報の網羅的検索 | HOKUTO（ガイドライン・薬剤DB） | 正面対決は不利。SEO経由で側面攻撃 |
| 医師同士のコミュニティ | ヒポクラ（本人確認済みSNS） | モデレーション負荷が個人開発で非現実的 |
| 転職・バイト紹介 | m3.com / 民間医局 | 営業力が必要。アフィリエイトで間接的に |
| 病院向けAI（B2B） | Ubie / AI問診 | B2Bは営業力とコンプラコストが高い |
| 包括的映像講義 | medu4 / Q-Assist | コンテンツ制作コストが高い。Studyで補完 |

## Perceptual Map

```
                    AI活用度 高
                        |
                  Ubie生成AI
                        |
              iwor ← ★空白地帯★
                        |
       ヒポクラ ─────────+───────── HOKUTO
                        |
              Antaa     |    MOTiCAN
                        |
              m3.com    |
                        |
                    AI活用度 低
     ←── 特定科/限定 ──────── 全科/汎用 ──→
```

iworのポジション: **「AI活用度が高い × 全科汎用の個人医師向けワークスペース」= 空白地帯**

- HOKUTO: 全科汎用だがAI非活用（情報参照ツール）
- Ubie: AI活用度高いがB2B（個人医師向けではない）
- MOTiCAN: AI非活用 × 特定科（内科専攻医限定）
- iworは「個人医師 × AI × 全科」の交差点を独占する

## 競合自動監視システム仕様

### アーキテクチャ

```
[Cloudflare Worker Cron（日次）]
  → PR TIMES RSS（医療カテゴリ）フェッチ
  → キーワードマッチ
  → KVに保存 + webhook通知（将来 Slack/Discord）
  → /api/competitors/alerts で閲覧
  → /admin/competitors で管理画面表示
```

### 監視対象キーワード

| カテゴリ | キーワード |
|---------|-----------|
| Tier 1 | MOTiCAN, HOKUTO, ホクト |
| Tier 2 | ヒポクラ, Antaa, アンター, m3, エムスリー |
| Tier 3 | Ubie, ユビー |
| 汎用 | 医師向けアプリ, 医師AI, J-OSLER, ヘルステック調達, 医療AI, 臨床支援 |

### 脅威レベル判定

| レベル | 条件 | 例 |
|--------|------|-----|
| Critical | Tier 1 + 「資金調達」「新機能」「買収」 | 「HOKUTO 30億円調達」 |
| High | Tier 1 のその他 or Tier 2 + 「資金調達」 | 「MOTiCAN Android対応」 |
| Medium | Tier 2 のその他 or 汎用キーワード + 「調達」 | 「医師AIスタートアップ5億調達」 |
| Low | 汎用キーワードのみ | 「医師向けアプリ比較記事」 |

### KV保存形式

```json
{
  "key": "competitor:alert:{timestamp}",
  "value": {
    "id": "alert_1711000000000",
    "title": "PR TIMESタイトル",
    "url": "https://prtimes.jp/...",
    "source": "prtimes",
    "matchedKeywords": ["HOKUTO", "資金調達"],
    "threatLevel": "critical",
    "competitor": "HOKUTO",
    "publishedAt": "2026-03-21T00:00:00Z",
    "fetchedAt": "2026-03-21T07:30:00Z",
    "dismissed": false
  },
  "expirationTtl": 7776000
}
```
