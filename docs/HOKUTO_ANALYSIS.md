- 特徴: 日本語対応、出典論文リンク、使用上の注意事項

### 2. 抗菌薬ガイド（60薬剤以上）
- βラクタム系（ペニシリン/セフェム1-4世代/カルバペネム）

- **iwor比較**: renal-dose-abx（20薬剤）で部分カバー。個別薬剤ページは未展開

### 3. ERマニュアル
- 症候一覧: 腹痛/頭痛/胸痛/腰痛/痺れ/めまい/失神/アナフィラキシー
- 外科系: 外傷初期診療/頭部外傷/頸部外傷/熱傷
- マイナーエマージェンシー: 鼻出血/釣り針除去
- 重要疾患: 消化器(GI出血/虫垂炎/胆嚢炎/胆管炎/膵炎)/呼吸器(肺炎/喘息/結核)/循環器(ACS/解離/PE/QT延長)/神経(脳梗塞/脳出血/SAH/TIA)/感染症(髄膜炎/UTI/IE/蜂窩織炎)

- 監修: 清水真人先生（聖路加救急部）

## iwor vs HOKUTO: 戦略的ポジショニング

### HOKUTOの強み
- 700+ツール（圧倒的な量）
- 専門医監修（信頼性）
- AI機能（検索/論文）
- 論文フィード（毎日更新）
- ユーザー基盤（医師の認知度高い）

### HOKUTOの弱み（＝iworのチャンス）
1. **アプリ内コンテンツがWeb検索に出ない** ← 最大
2. ERマニュアルはテキストベース（インタラクティブではない）
3. 生活習慣病の総合管理ツールがない
4. J-OSLER/専攻医特化コンテンツがない
5. 計算ツールは入力→結果のみ（次のアクション提示がない）

### iworの差別化戦略
1. **Web-first**: 全コンテンツをSEOで拾える状態に（HOKUTO最大の盲点）
2. **アクション提示型**: 計算結果だけでなく「次に何をすべきか」を提示
3. **インタラクティブフロー**: 血ガス解釈/主訴別ERアプローチをステップバイステップで
4. **ニッチ特化**: J-OSLER/専攻医/内科外来 → HOKUTOがカバーしない領域
5. **生活習慣病 総合管理**: HOKUTOにない外来支援ツール
