## ポジショニング

### 一言で
> 「HOKUTOは医師のアテンションを製薬に売っている。iworは医師に気持ちいい体験を売っている。」

### アナロジー
- HOKUTO = Gmail（無料、広告あり、機能的）
- iwor = Superhuman（有料、広告ゼロ、体験に対価を払う層が存在する）
- Linearが$35Kのマーケティング費用で$1.25B企業になったのと同じ構造

### 差別化軸（HOKUTOにないもの）
1. **Study**（FSRS暗記カード）— HOKUTOは学習機能なし
2. **J-OSLER管理** — HOKUTOはJ-OSLER未対応
3. **マッチング対策** — HOKUTOはキャリア支援なし
4. **広告ゼロ** — HOKUTOは製薬広告が収益の柱
5. **体験設計** — 疲弊した脳に負荷をかけないUI
6. **Webファースト** — HOKUTOはアプリ囲い込み。Web版は最近ローンチでSEO弱い。iworはGoogle検索で先に見つかる

### 競合の具体的成長戦略（リサーチ結果）

| 競合 | 成長の核 | iwor応用可能性 |
|------|---------|--------------|
| HOKUTO | 創業者が全82医学部を個人訪問→医学生アプリ→卒後に本アプリ転換 | 医学生パイプライン戦略は正しい。SEOで代替 |
| MOTiCAN | 医師個人開発者(@digista_net)がXで380K impressions→45日で1,000ユーザー | **直接の成功モデル**。X運用が最優先 |
| m3.com | ポイント経済（製薬メール開封で月¥5,000-20,000稼げる）→93.4億円収益 | ポイントは製薬資金なので真似不可。体験で差別化 |
| Antaa | 2年間の対面勉強会→コロナでオンライン化→7万人 | 信頼構築が先、スケールが後 |
| Anki(AnKing) | 「ゴールドスタンダードデッキ」統合→30万DL。デッキ=配布装置 | **日本版AnKing戦略**: 内科専門医ゴールドデッキ作成→iwor Study配布 |
| Q-Assist | 学年代表経由の共同購入→3年60%/4年80%/5-6年70% | 大学・プログラム単位のグループPRO購入検討 |
| medu4 | 個人講師ブランド→2025年サービス終了（YouTube無料化で崩壊） | 単一クリエイター依存は危険。プラットフォーム型に |

## 獲得戦略（3チャネル集中）

### 原則
- **予算ゼロ**。時間だけが投資資源。
- **3チャネルに集中**。分散は死。
- **PLG（Product-Led Growth）**。プロダクト自体がマーケティング。

### Channel 1: SEO（長期エンジン）

**現状**: 173記事、166計算ツール。既にアセットは大量にある。

**戦略**:
- BOFU（Bottom of Funnel）キーワードに集中
  - 「J-OSLER 病歴要約 書き方」「内科専門医試験 勉強法」「マッチング 病院 選び方」
  - これらで検索する人 = iworのペルソナそのもの
- 既存173記事の内部リンク最適化（孤立ページ解消）
- 各計算ツールページからStudyへの導線整備
- 新規記事: 月4本ペース（週1）

**タイムライン**: 3-6ヶ月で初期成果、12ヶ月で本格的なトラフィック

### Channel 2: X（Twitter）（中期バイラル）

**現状**: アカウント未運用

**戦略**:
- **個人アカウント**で運用（企業アカウントは信頼されない）
- 投稿カテゴリ:
  1. 臨床Tips（計算ツールの使い方、知識共有）— 50%
  2. 開発ログ（Build in Public）— 20%
  3. Study/J-OSLER Tips — 20%
  4. プロダクト更新 — 10%
- **週5投稿**目標
- 医クラ（#医クラ #研修医 #専攻医 #JOSLER #内科専門医試験）ハッシュタグ活用
- **ストリークシェアカード**: Study達成時にX投稿用画像を自動生成 → iwor branding付き
- 学会シーズン（3-4月）にライブツイート → 学会まとめ → 論文フィード誘導

**日本の医師X利用率: 58.7%** — 最もリーチしやすいチャネル

### Channel 3: 先輩→後輩 口コミ（最強チャネル）

**現状**: 未着手

**戦略**:
- **Micro-KOLプログラム**: 主要教育病院の研修医/専攻医10人に無料PRO提供
  - 条件: 正直なフィードバック + 気が向いたらXで言及
  - KOLに必要なのは10万フォロワーではなく「同じ病院の200人の後輩に影響力がある人」
- **当直シフト共有**: 参加者リンクにiworのさりげないCTA → バイラル装置
- **Referralプログラム**: 紹介した人もされた人もPRO 1ヶ月無料
  - Dropbox: 紹介で3,900%成長（15ヶ月で10万→400万）
  - 紹介ユーザーは通常の1.8倍リテンション、1.25倍課金
- **医局文化の活用**: 1人が使い始めると同じ科の全員に広まる（医局内情報伝播）

## 信頼構築（Trust Signals）

日本の医師が信頼するもの:
1. **先輩の推薦**（最強）→ Micro-KOL + 口コミ
2. **学会での認知** → 学会シーズンのX活動
3. **大学教授の推薦** → 将来的にアプローチ
4. **出典の明記** → 全ツールにSourceCitation済み
5. **美しいデザイン** → 「ちゃんとしてる」印象 → 既に達成

日本の医師が嫌うもの:
1. 派手なマーケティング → やらない
2. 強引なセールス → やらない
3. 患者情報リスク → SaMD回避済み
4. 実名強制 → 匿名OK設計

- インフルエンサーマーケティング（有料）— 予算なし
- 学会ブース出展 — コストに見合わない
- 「日本初」等の検証不能な主張 — 炎上リスク
