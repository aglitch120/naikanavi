## B. 新アプリ追加

### B-1. コミュニティ機能（/community/）— PRO 🟠

医師・医学生向けSNS。無料会員にはダミー投稿（興味をそそる内容）を表示してPRO誘導。

**基本機能:**
- スレッド投稿（テキスト）
- タグ付け（デフォルトタグ候補: 当直、ふるさと納税、キャリア、マッチング対策、初期研修、USMLE、バイト、マネー、人間関係、プチコンサル、iworにほしい機能 など）
- 好きなトピックでフィルタ（キャリア、マネー、研修、人間関係、プチコンサル、iworにほしい機能）
- 返信機能（スレッド形式）
- いいね / Upvote・Downvote（Reddit風評価）
- 検索機能、ソート（最新/人気/トレンド）
- ブックマーク / お気に入り保存
- 通報（Report）機能
- アンケート（投票）機能
- 編集・削除機能（本人 + モデレーター）
- レートリミット（1分3投稿制限）

**アカウント・プロフィール:**
- プロフィール画像: 人型アイコン + かわいいマーク10種類から選択
- コミュニティ表示情報: ざっくり年代、性別、病院規模、専門医（希望時のみ）
- 回答いいね数でランクアップ機能

**通知システム:**
- 返信通知、メンション（@ユーザー名）通知
- Web通知 + メール通知（PWA対応）
- デフォルトON → 設定でOFF可能
- 全面的な宣伝は避ける

**モデレーション:**
- 投稿ガイドライン自動レビュー（攻撃的発言、誹謗中傷を自動弾き）
- モデレーターによる強制削除・編集権限

**SEO:**
- SSR/SSG対応（検索エンジン向け）
- 動的OGP画像生成
- XMLサイトマップ自動更新

**FREE/PRO線引き:**
- FREE: ダミー投稿表示（リッチなモザイクteaser）
- PRO: 全機能利用可能

### B-2. 当直シフト作成（/shift/）— FREE（マーケティング装置）🟠

HOKUTO「当直シフト作成AI」を参考（完全模倣は避ける）。
**iworの強み: アプリダウンロード不要、ブラウザで完結。**

**フロー（ステップウィザード）:**
1. シフトグループ作成（病院名 + シフト表名称）
2. 対象月の選択（今月/来月/再来月/その他）
3. 医師の登録（名前リスト、後から追加・削除可能）
4. シフト枠の設定（カレンダーUI、平日当直/休日当直/休日日直/祝日夜勤 等）
5. ルール設定（各枠に入れる医師の選択、連続勤務間隔、勤務量バランススライダー）
6. 希望日入力（医師×日付マトリクス、◎/○/△/× の4段階）
7. AI自動シフト割り当て（Cloudflare Workers AI or ルールベース）
8. 結果表示（カレンダー + 医師別割当状況、手動調整可能）
9. 印刷・共有

**共有機能（バイラルマーケティング）:**
- メール/LINEでリンク送信ボタン（パスワード設定可能）
- 「このリンクをコピーして共有」ボタン
- 回答期限設定
- 参加者の回答入力画面にiworのさりげないCTA表示

**FREE理由:** ユーザーが勝手にリンクを広めてくれる → 無料の最強マーケティング

### B-3. 学会スケジュール & リマインダー（/conferences/）— FREEMIUM 🟠

医師・医学生の学会参加を一元管理するアプリ。

**学会データベース:**
- **専門医機構認定の基本領域19学会 + サブスペシャルティ領域24学会 = 必須掲載**
- その他規模が大きい学会（参加者500名以上目安）も全掲載
- 各学会の情報: 正式名称、略称、開催日程、会場、都市、抄録締切日、参加登録締切日、参加費、公式サイトURL、対象診療科タグ
- 年次大会 + 地方会 + 支部会 もカバー（データが取れる範囲で）
- 運営側で定期的にデータ更新（将来的にはスクレイピング/API自動化）

**ユーザー機能:**
- 診療科・興味分野でフィルタ（自分に関連する学会だけ表示）
- カレンダービュー（月/週表示）+ リスト表示 切替
- 「参加予定」ボタンでマイリスト登録
- マイリストから自分の学会スケジュール一覧
- 各学会の詳細ページ（日程・会場・アクセス・締切一覧）

**通知機能（PWA対応）:**
- 登録した学会の抄録締切 X日前リマインド
- 参加登録締切 X日前リマインド
- 開催 X日前リマインド
- 通知タイミングはユーザー設定可（7日前/3日前/1日前/当日）
- Web Push通知 + メール通知
- デフォルトON → 設定でOFF可能

**FREE/PRO線引き:**
- FREE: 学会DB閲覧 + カレンダー表示 + 3学会までマイリスト登録
- PRO: 無制限マイリスト + 全通知機能 + 抄録締切カウントダウン + iCalエクスポート
