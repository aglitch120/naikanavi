## 内科専門医試験とは？制度の全体像

**内科専門医試験**は、日本内科学会が主催する内科専門医の認定試験です。2019年から新専門医制度に基づく試験が開始され、J-OSLER修了者が受験できます。

### 旧制度との違い

| 項目 | 旧制度（認定内科医） | 新制度（内科専門医） |
|------|------------------|------------------|
| 前提 | 医師免許取得後3年以上 | J-OSLER修了 |
| 試験形式 | 筆記 | CBT（コンピュータ） |
| 試験範囲 | 内科全般 | 内科全般＋症例問題 |
| 合格率 | 80〜90%台 | 85〜95%台 |

新制度では**CBT形式**（Computer-Based Testing）が採用され、マークシートではなくパソコン画面での受験になりました。

### 内科専門医取得のメリット

**キャリア面**での最大のメリットは、**サブスペシャリティ専門医（循環器・消化器など）の受験資格**を得られることです。多くのサブスペシャリティ学会が「内科専門医取得」を受験要件としています。

また、**給与・待遇面**でも、専門医の有無で基本給や手当が変わる病院は少なくありません。

## 受験資格と申し込み手順

### 受験資格の確認

以下の条件をすべて満たす必要があります。

1. **医師免許を有すること**
2. **内科専攻医プログラムを修了していること**（J-OSLER修了認定）
3. **日本内科学会の会員であること**

J-OSLERの修了認定は例年**3〜4月**に行われます。修了認定を受けた年の秋の試験から受験可能です。

### 申し込みの流れ

```
1. 日本内科学会のWebサイトで受験申込み（例年6〜7月）
2. 受験料の支払い（約30,000円）
3. 受験票の受領（試験1〜2週間前）
4. 試験会場での受験（例年11月）
5. 合否通知（例年12〜1月）
```

**申し込みの締切を絶対に逃さないことが重要**です。試験は年1回しか実施されないため、申し込み忘れは1年間の遅れに直結します。

→[内科専門医試験の申し込み方法・日程完全ガイド](/blog/g09-naika-senmoni-moshikomi)で詳しい手順を確認してください。

## 試験の形式・難易度・合格率

### 試験形式

- **形式**: CBT（コンピュータ画面で選択肢を選ぶ）
- **問題数**: 約200問
- **試験時間**: 約4時間（途中休憩あり）
- **出題形式**: 単純選択・複数選択・臨床症例問題

臨床症例問題では、患者情報・検査値・画像などが提示され、診断・治療を問われます。**暗記だけでなく、臨床的な思考力が問われる問題**が含まれる点が国試と異なります。

### 難易度

**国試と比べてどう違うか？**

- 範囲は内科全般で国試と重なる部分も多い
- ただし**より臨床的・実践的な問題**が多い
- 「教科書通りの答え」ではなく「実際の臨床現場での判断」を問う問題がある
- 合格率は85〜95%台と高く、**きちんと対策すれば合格できる試験**

→[内科専門医試験の難易度は？国試との違いと対策ポイント](/blog/g07-naika-senmoni-nanido)で詳細な難易度分析を確認できます。

### 合格率データ

| 回 | 年度 | 合格率 |
|----|------|--------|
| 第1回 | 2019年 | 約93% |
| 第2回 | 2020年 | 約91% |
| 第3回 | 2021年 | 約85% |
| 第4回 | 2022年 | 約90% |
| 第5回 | 2023年 | 約95% |

第3回で合格率が下がった背景には、問題の難化があったと言われています。ただし、全体として**しっかり対策すれば85〜95%の合格率**を維持しています。

## 領域別の出題傾向と対策

内科専門医試験は**内科13領域**から出題されます。領域ごとに出題数・難易度が異なるため、**戦略的な学習配分**が合格への近道です。

### 出題数の多い領域（重点対策推奨）

**循環器内科**  
心不全・不整脈・虚血性心疾患からの出題が多い。ガイドラインの治療方針を押さえることが重要です。→[循環器内科の試験対策](/blog/h01-naika-senmoni-junkanki)

**消化器内科**  
炎症性腸疾患・肝硬変・消化管腫瘍が頻出。内視鏡所見の読み方も問われます。→[消化器内科の試験対策](/blog/h02-naika-senmoni-shoukaki)

**呼吸器内科**  
肺炎・COPD・肺癌・間質性肺炎が頻出。画像所見（胸部X線・CT）の問題が多い傾向があります。→[呼吸器内科の試験対策](/blog/h03-naika-senmoni-kokyuuki)

### 比較的対策しやすい領域

**内分泌・代謝**  
糖尿病・甲状腺疾患が多く、治療薬の選択肢を正確に覚えることが得点につながります。→[内分泌・代謝の試験対策](/blog/h04-naika-senmoni-naibunpitsu)

**腎臓内科**  
CKD・電解質異常・腎炎が中心。計算問題（GFR・電解質補正）も出題されます。→[腎臓内科の試験対策](/blog/h05-naika-senmoni-jinzou)

### 難易度が高い領域

**血液内科・神経内科**  
専門性が高く、サブスペシャリティをもたない専攻医には難しい問題が多い。**完璧を目指さず、基本問題を確実に取る戦略**が有効です。→[血液内科の試験対策](/blog/h06-naika-senmoni-ketsueki) / [神経内科の試験対策](/blog/h07-naika-senmoni-shinkei)

### 領域別学習の優先順位

```
最優先：循環器・消化器・呼吸器（出題数多・得点源）
　次点：内分泌・腎臓・感染症（比較的対策しやすい）
余力で：血液・神経・膠原病（難易度高・基本問題に絞る）
```

## 合格者が実践した勉強法

### いつから始めるべきか

**J-OSLER修了認定（3〜4月）から試験（11月）まで約7ヶ月**あります。

- **理想**: 修了認定後すぐに開始（7ヶ月計画）
- **現実的な最短**: 試験3ヶ月前（8月）から本格開始でも合格可能
- **ギリギリ**: 1ヶ月前からは厳しい（ただし基礎力がある場合は可能）

**3ヶ月計画の目安**

```
1〜2ヶ月目（8〜9月）：参考書で全領域を一周
3ヶ月目（10月）：過去問演習・弱点補強
試験直前（11月直前）：頻出事項の最終確認
```

→[内科専門医試験の勉強法｜いつから・何を・どのくらい](/blog/g01-naika-senmoni-benkyouhou)で詳細なスケジュールを解説しています。

### 効果的な学習戦略

**過去問中心の学習が最も効率的**  
出題傾向は年によってある程度パターン化されています。過去問を繰り返し解き、「なぜその選択肢が正解か」を理解することが合格への最短ルートです。

**弱点領域を早期に特定する**  
模擬試験や問題集で得点できない領域を1〜2ヶ月目に特定し、重点的に学習します。試験直前に弱点発見しても手遅れになりがちです。

**ガイドライン改訂に注意する**  
内科専門医試験は**最新のガイドライン**に基づいた問題が出題されます。参考書の出版年度を確認し、古いガイドラインに基づく知識がないかチェックしてください。

### 忙しい専攻医のための時間術

3年目専攻医は病院業務が多忙な時期でもあります。隙間時間の活用が合否を分けます。

- **通勤時間**: スマホアプリで問題演習（1日10〜20問）
- **当直の空き時間**: 参考書1章分の精読
- **休日**: まとめて3〜4時間の集中学習

**毎日30分でも継続できれば、3ヶ月で十分な学習量を確保できます。**

通称「朝倉」。内科のバイブルで、試験範囲の全領域を網羅。ただし内容が膨大なため、通読より「辞書的な使い方」が現実的です。

医学生時代に使った人も多く、馴染みやすい。ただし国試対策向けの内容が多いため、専門医試験の臨床問題には別途対策が必要です。

## 試験当日の流れと注意点

### 試験会場と試験の流れ

内科専門医試験はCBT形式のため、全国の**プロメトリックテストセンター**（またはそれに準ずる会場）で受験します。

```
受付（試験開始30分前まで）
↓
本人確認・持ち物検査
↓
試験（約4時間、途中休憩あり）
↓
終了・退場
```

→[内科専門医試験 当日の流れと注意点](/blog/g10-naika-senmoni-tojitsu)で詳細な当日の流れを確認できます。

### 持ち物・注意事項

- **受験票**（必須）
- **身分証明書**（顔写真付き）
- **時計**（スマートウォッチ不可、会場にも時計あり）
- 電卓は**不要**（CBT内で使用可能）
- 参考書・メモは**持ち込み不可**

**試験当日に特に注意したいポイント**  
CBT会場のパソコン操作に慣れていない場合、操作に戸惑って時間をロスするリスクがあります。事前に**CBTのデモンストレーション**を確認しておくことをおすすめします（プロメトリックの公式サイトで体験可能）。

## 不合格になったら？再受験の戦略

合格率85〜95%の試験とはいえ、**5〜15%の受験者は不合格**になります。不合格の場合の対応を事前に知っておくことが重要です。

### 再受験のルール

- **再受験は翌年度の試験**から可能
- J-OSLERの修了認定は維持される（再申請不要）
- 受験料は再度支払いが必要

### 不合格後にやるべきこと

**1. 自分の得点開示を確認する**  
日本内科学会から得点フィードバックがある場合、どの領域が弱かったかを把握します。

**2. 弱点領域を徹底的に補強する**  
不合格の原因になった領域を特定し、翌年の試験まで重点的に学習します。

**3. 勉強方法自体を見直す**  
「暗記中心だった」「過去問演習が不足していた」など、勉強法に問題があった可能性もあります。

→[内科専門医試験に不合格になったら｜再受験の対策と心構え](/blog/g11-naika-senmoni-fugoukaku)で詳しい再受験戦略を解説しています。

## 総合内科専門医との違い

**内科専門医**と**総合内科専門医**は別の資格です。混同している方も多いため、違いを整理します。

| 項目 | 内科専門医 | 総合内科専門医 |
|------|----------|--------------|
| 主催 | 日本内科学会 | 日本内科学会 |
| 受験資格 | J-OSLER修了 | 内科専門医取得後5年以上 |
| 難易度 | 合格率85〜95% | 合格率60〜70%台 |
| キャリア | サブスペシャリティへの入口 | ジェネラリストとして活躍 |

**内科専門医はサブスペシャリティへの「入口」**、**総合内科専門医はジェネラリストとしての「完成形」**というイメージです。

総合内科専門医については→[総合内科専門医 完全ガイド](/blog/i08-sougou-naika-senmoni-guide)で詳しく解説しています。

## まとめ：合格へのロードマップ

内科専門医試験で合格するためのポイントをまとめます。

**受験前の準備**
- J-OSLER修了認定の確認（3〜4月）
- 日本内科学会への受験申し込み（6〜7月）
- 学習計画の立案（修了認定後すぐに開始が理想）

**学習戦略**
- 過去問中心の学習で出題傾向を把握
- 循環器・消化器・呼吸器を重点的に対策
- 最新ガイドラインの改訂に注意
- 忙しい専攻医こそ隙間時間の積み重ね

**試験直前・当日**
- CBT操作に事前に慣れておく
- 受験票・身分証明書の準備
- 当日は時間配分を意識する

---

内科専門医試験は、適切な準備をすれば**確実に合格できる試験**です。J-OSLERを修了した皆さんなら、その努力は本物です。あとは試験対策を戦略的に進めるだけです。

内科専門医取得後のキャリアについては→[内科専攻医のキャリア設計ロードマップ](/blog/n01-naika-career-path)で解説しています。ぜひ合わせてご覧ください。

## よくある質問（FAQ）


**Q. 内科専門医試験はいつ実施されますか？**

例年11月に実施されます。申し込みは例年6〜7月です。具体的な日程は毎年日本内科学会の公式サイトで発表されます。詳しくは[内科専門医試験の申し込み方法・日程](/blog/g09-naika-senmoni-moshikomi)をご確認ください。



**Q. J-OSLER修了後、すぐに受験できますか？**

修了認定を受けた年の秋の試験から受験可能です。3年目で修了認定を受けた場合、その年の11月に初めて受験できます。



**Q. 試験に落ちたら、J-OSLERをやり直す必要がありますか？**

J-OSLERの修了認定は維持されます。再受験の際は翌年度の試験に申し込むだけで、J-OSLERのやり直しは不要です。



**Q. 内科専門医と総合内科専門医、どちらを取るべきですか？**

まず内科専門医を取得してください。総合内科専門医は内科専門医取得後5年以上経過しないと受験できません。サブスペシャリティを目指す方も、ジェネラリストを目指す方も、まず内科専門医が必須です。



**Q. 働きながら合格できますか？**

合格者の多くが働きながら合格しています。1日30分の学習を3〜6ヶ月継続することが現実的な目標です。隙間時間を活用したスマホアプリ学習が特に効果的です。
