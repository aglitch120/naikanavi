## F. マッチング対策 大改修（`/matching/` → マッチング・転職対策）

### F-1. 名称・構造変更
- 名称: 「マッチング対策」→「**マッチング・転職対策**」
- モード切替: 医学生（初期研修マッチング）モード / それ以外（専攻病院・転職）モード
  - カチッとトグルで切替
  - 会員登録時のプロフィールでデフォルト自動設定

### F-2. タブ構造（順序）
1. **プロフィール** → 履歴書生成
2. **書類・メール生成**（履歴書以外: 見学申込メール、お礼メール、採用御礼メール、送付状等）
3. **病院データベース**（マッチングモード時のみ表示）
   - 一番上: 「あなたに最適な病院リスト」（モダンなダミーモザイク、teaser、高級感）
4. **志望リスト**（マッチングモード時のみ表示）
   - 一番上: マッチ成功率teaser（最大99%、100%にしない）
5. **AI面接**
   - 下部: フィードバックのリッチなモザイクteaser

### F-3. プロフィール入力の大幅改善
- 「10分ほどかかります」の案内表示
- 基本はクリック / 候補選択 / ドロップダウン / (その他)短文入力
- 質問に答えるだけで埋まるウィザード形式

- ライフスタイル（勤務スタイル/年収/WLB/家族計画）
- 地域・環境（地域タイプ/条件/職場環境/通勤）
