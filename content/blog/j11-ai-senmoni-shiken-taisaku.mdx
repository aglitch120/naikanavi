## 実践的なAI活用プロンプト集

### 活用法1：問題を作ってもらう

```
【プロンプト例】
内科専門医試験向けに、心不全の治療に関するMCQ問題を5問
作成してください。難易度は中級程度で、選択肢は5択にして
ください。正答と解説も一緒に出してください。
```

「内科専門医試験レベル」と明示することで適切な難易度の問題が作られます。出題テーマは「循環器」「消化器」「腎臓」など領域別に指定すると効率的です。

### 活用法2：苦手な疾患を深掘りする

```
【プロンプト例】
慢性腎臓病のステージ分類（G分類）と蛋白尿のA分類を、
臨床的な意味を含めてわかりやすく説明してください。
各ステージで特に注意すべき合併症と治療介入のポイントも
教えてください。
```

教科書を読んで理解できなかった部分を具体的に質問することで、理解が格段に深まります。

### 活用法3：鑑別診断を練習する

```
【プロンプト例】
40代男性、突然の胸痛で救急来院。以下のバイタルと検査
結果を提示します。[バイタル・検査値を入力]
主な鑑別診断と、各疾患を支持・否定する所見を整理して
ください。
```

J-OSLERの症例をベースに鑑別診断を練習することで、病歴要約の総合考察の質も同時に向上します。

### 活用法4：間違えた問題の解説を深める

```
【プロンプト例】
次の問題を間違えました。正答はBですが私はCを選びました。
なぜBが正解でCが誤りなのか、病態生理から詳しく説明して
ください。[問題文を貼り付ける]
```

間違えた問題をそのまま貼り付けて「なぜ間違えたか」を徹底的に分析してもらうことで、類似問題でのミスが減ります。

### 活用法5：まとめノートを作る

```
【プロンプト例】
内科専門医試験で出やすい「電解質異常（低Na血症・高K血症）」
の重要ポイントを、表形式でまとめてください。
原因・症状・治療の3列で作成してください。
```

## AI活用の3つの注意点

### 注意1：医学的正確性を必ず確認する

AIは誤った医学情報を自信満々に提示することがあります（ハルシネーション）。AIの回答は**必ず教科書・ガイドラインで確認する習慣**をつけましょう。特に数値（用量・検査基準値・生存率）は必ずダブルチェックが必要です。

### 注意2：AIは補助ツールとして位置づける

AIはあくまで補助ツールです。理解の深掘り・問題演習の多様化に使い、**メインの学習は教科書・問題集を軸**にすることが重要です。AIだけに頼ると基礎知識が抜け落ちるリスクがあります。

### よくある失敗：患者情報をAIに入力する

AIで学習する際、実在する患者の情報（氏名・ID・詳細な病歴）を入力することは**絶対に禁止**です。個人情報保護法違反・医師法違反のリスクがあります。症例を練習に使う場合は必ず匿名化・脱識別化した上で使用してください。

## まとめ

AIを試験対策に活用する際は、以下の5つの用途から始めましょう。

**AI活用の5用途**：問題作成・苦手深掘り・鑑別練習・間違い分析・まとめノート作成

AIの回答は**必ず教科書・ガイドラインで正確性を確認**する習慣とセットで活用してください。

---


AI活用は「楽をする」のではなく「学習の質を上げる」ためのツールです。弱点分析・問題作成・解説生成という3つの用途を組み合わせることで、限られた勉強時間を最大化できます。ただし、AIの出力を鵜呑みにせず、必ず公式テキストや信頼できる文献で確認する習慣を忘れないようにしてください。AIは優秀なアシスタントですが、最終的な判断は自分の頭で行うことが専門医としての基礎力を養います。

## よくある質問

**Q. AI生成の問題は実際の試験と同じレベルですか？**
完全に同じではありませんが、考え方の練習として有効です。公式の過去問と組み合わせて使うことを推奨します。

**Q. AIに個人情報（患者情報）を入力してもよいですか？**
絶対に入力しないでください。患者情報は匿名化・脱識別化した上でAIを使用してください。

**Q. 無料版のChatGPTで試験対策は十分ですか？**
無料版でも問題作成・解説の依頼は可能です。ただし回答の質や速度が制限されるため、本格的に使うなら有料版が推奨されます。
